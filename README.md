# Blog Platform with Comments
A full-stack blogging application built using the MERN Stack.

## Features
- User Authentication (JWT)
- Create/Edit/Delete Blogs
- Comment System
- MongoDB Database
- REST APIs
- Responsive UI

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT
- Bootstrap/CSS

## Installation
git clone <repository-url>
cd server
npm install
npm start
cd client
npm install
npm start
